# Data Structure & Algorithms Tasks

Este repositório foi desenvolvido com finalidade acadêmica e de estudo, focado exclusivamente no treinamento de lógica de programação, análise de algoritmos e implementação de estruturas de dados. 

The main goal of this project was to consolidate fundamental computer science concepts in a practical way, explicitly covering the following topics:

- **Complexity Analysis**: Concepts and application of Big O Notation.
- **Algorithms**: Recursion and Sorting algorithms implementations.
- **Data Structures**: 
  - Arrays
  - Graphs
  - Hash Tables
  - Linked Lists
  - Stacks and Queues
  - Trees

## Getting Started

1. **Clone the repository**: `git clone git@github.com:JVGirardi/dsa-tasks.git`
2. **Navigate to the project directory**: `cd dsa-tasks`
3. **Compile the desired file**: `javac FileName.java`
4. **Run the algorithm**: `java FileName`
